Bootstrap Cheat Sheet

Сетка (Grid)

  • .container - Создает фиксированный контейнер с отступами по краям.
  • .container-fluid - Создает контейнер на всю ширину экрана.
  • .row - Создает строку для размещения колонок.
  • .col - Колонка, которая автоматически подстраивается под ширину.
  • .col-{size}-{number} - Определяет ширину колонки (например, .col-md-4 для средней ширины 4).
  • .offset-{size}-{number} - Добавляет отступы слева от колонки (например, .offset-md-4).

Кнопки (Buttons)

  • .btn - Основной класс кнопки.
  • .btn-primary - Синяя кнопка (основная).
  • .btn-secondary - Серая кнопка (второстепенная).
  • .btn-success - Зеленая кнопка (успешная).
  • .btn-danger - Красная кнопка (опасная).
  • .btn-warning - Желтая кнопка (предупреждение).
  • .btn-info - Голубая кнопка (информация).
  • .btn-light - Светлая кнопка.
  • .btn-dark - Темная кнопка.
  • .btn-link - Кнопка-ссылка.
  • .btn-lg - Большая кнопка.
  • .btn-sm - Маленькая кнопка.
  • .btn-block - Кнопка на всю ширину контейнера.

Формы (Forms)

  • .form-control - Основной класс для полей ввода.
  • .form-group - Контейнер для группировки элементов формы с отступами.
  • .form-check - Контейнер для чекбоксов и радиокнопок.
  • .form-check-input - Класс для чекбоксов и радиокнопок.
  • .form-check-label - Класс для меток чекбоксов и радиокнопок.
  • .form-inline - Размещает элементы формы в одну линию.
  • .input-group - Контейнер для объединения полей ввода с дополнительными элементами (например, кнопками или текстом).
  • .input-group-prepend - Контейнер для элементов, предшествующих полю ввода.
  • .input-group-append - Контейнер для элементов, следующих за полем ввода.
  • .input-group-text - Класс для добавления текста или элементов внутри .input-group.

Карточки (Cards)

  • .card - Основной контейнер для карточки.
  • .card-body - Контейнер для основного содержимого карточки.
  • .card-title - Класс для заголовка карточки.
  • .card-text - Класс для текста внутри карточки.
  • .card-img-top - Класс для изображения в верхней части карточки.
  • .card-footer - Класс для нижней части карточки.

Навигационная панель (Navbar)

  • .navbar - Основной класс для навигационной панели.
  • .navbar-expand-lg - Расширяет панель на больших экранах.
  • .navbar-light - Светлая тема навигационной панели.
  • .navbar-dark - Темная тема навигационной панели.
  • .navbar-brand - Класс для логотипа или названия сайта.
  • .navbar-toggler - Кнопка для сворачивания навигационной панели на маленьких экранах.
  • .navbar-collapse - Контейнер для элементов, которые сворачиваются.
  • .nav-item - Класс для элемента навигации.
  • .nav-link - Класс для ссылки в элементе навигации.

Модальные окна (Modals)

  • .modal - Основной класс для модального окна.
  • .modal-dialog - Контейнер для содержимого модального окна.
  • .modal-content - Контейнер для содержимого модального окна.
  • .modal-header - Класс для заголовка модального окна.
  • .modal-body - Класс для содержимого модального окна.
  • .modal-footer - Класс для нижней части модального окна.
  • .modal-title - Класс для заголовка модального окна.
  • .close - Класс для кнопки закрытия модального окна.

Утилиты (Utilities)

  • .text-center - Выравнивание текста по центру.
  • .text-right - Выравнивание текста по правому краю.
  • .text-left - Выравнивание текста по левому краю.
  • .text-muted - Приглушенный цвет текста.
  • .bg-primary - Синий фон.
  • .bg-secondary - Серый фон.
  • .bg-success - Зеленый фон.
  • .bg-danger - Красный фон.
  • .bg-warning - Желтый фон.
  • .bg-info - Голубой фон.
  • .bg-light - Светлый фон.
  • .bg-dark - Темный фон.
  • .p-1 - Отступы (padding) размером 0.25rem.
  • .p-2 - Отступы (padding) размером 0.5rem.
  • .p-3 - Отступы (padding) размером 1rem.
  • .m-1 - Поля (margin) размером 0.25rem.
  • .m-2 - Поля (margin) размером 0.5rem.
  • .m-3 - Поля (margin) размером 1rem.
