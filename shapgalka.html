Шпаргалка по CSS для адаптивного дизайна

Медиа-запросы

Используются для применения различных стилей в зависимости от характеристик устройства (например, ширина экрана).

@media (min-width: 600px) and (max-width: 800px) {
    /* Стили для экранов от 600px до 800px */
}

Flexbox

display: flex

Определяет элемент как flex-контейнер, внутри которого дочерние элементы автоматически становятся flex-элементами.

.container {
    display: flex;
}

flex-direction

Устанавливает направление главной оси (горизонтально или вертикально).

.container {
    flex-direction: row; /* Элементы располагаются в строку */
    flex-direction: column; /* Элементы располагаются в колонку */
}

flex-wrap

Определяет, должны ли элементы переноситься на новую строку или колонку, если они не умещаются.

.container {
    flex-wrap: nowrap; /* Все элементы в одной линии */
    flex-wrap: wrap; /* Элементы переносятся на новую линию */
}

justify-content

Выравнивание элементов вдоль главной оси.

.container {
    justify-content: flex-start; /* По началу контейнера */
    justify-content: center; /* По центру */
    justify-content: space-between; /* Равномерно с пространством между элементами */
}

align-items

Выравнивание элементов вдоль поперечной оси.

.container {
    align-items: flex-start; /* По началу поперечной оси */
    align-items: center; /* По центру поперечной оси */
    align-items: stretch; /* Растягивание элементов на всю высоту контейнера */
}

align-content

Выравнивание линий элементов вдоль поперечной оси, если элементы переносятся.

.container {
    align-content: flex-start; /* Линии выравниваются по началу поперечной оси */
    align-content: center; /* Линии выравниваются по центру поперечной оси */
    align-content: space-between; /* Равномерное распределение линий с пространством между ними */
}

flex

Гибкая величина для элементов flex-контейнера.

.element {
    flex: 1; /* Элемент будет растягиваться, занимая доступное пространство */
}

order

Определяет порядок отображения элементов внутри flex-контейнера.

.element1 {
    order: 1; /* Этот элемент будет отображен первым */
}
.element2 {
    order: 2; /* Этот элемент будет отображен вторым */
}

Размеры

width и height

Устанавливают ширину и высоту элементов.

.element {
    width: 100%; /* Ширина элемента 100% от родительского контейнера */
    height: auto; /* Высота элемента авто подстраивается */
}

max-width и min-width

Устанавливают максимальную и минимальную ширину элементов.

.element {
    max-width: 600px; /* Максимальная ширина элемента 600px */
    min-width: 300px; /* Минимальная ширина элемента 300px */
}

max-height и min-height

Устанавливают максимальную и минимальную высоту элементов.

.element {
    max-height: 600px; /* Максимальная высота элемента 600px */
    min-height: 300px; /* Минимальная высота элемента 300px */
}

Типы отображения

display

Определяет тип отображения элемента.

.element {
    display: block; /* Блочный элемент */
    display: inline; /* Встроенный элемент */
    display: flex; /* Flex-контейнер */
}
