CSS Grid Layout Properties

grid-template-columns

Определяет количество и размеры колонок.

grid-template-columns: 100px 200px 1fr;

grid-template-rows

Определяет количество и размеры строк.

grid-template-rows: auto 100px 1fr;

grid-gap

Устанавливает расстояние между колонками и строками.

grid-gap: 10px 15px;

grid-auto-rows

Определяет высоту автоматически добавляемых строк. В данном примере устанавливает минимальную высоту строки в 100px, но позволяет ей расширяться, если это необходимо.

grid-auto-rows: minmax(100px, auto);

grid-template-areas

Позволяет именовать области сетки и размещать элементы по именам.

grid-template-areas:
"header header header"
"sidebar content content"
"footer footer footer";

grid-area

Сокращённая запись для задания области сетки, которую занимает элемент.

grid-area: 1 / 2 / 4 / 5;

grid-row-start

Указывает начальную строку для элемента.

grid-row-start: 2;

grid-column-start

Указывает начальную колонку для элемента.

grid-column-start: 1;

grid-column

Указывает начальную и конечную колонки для элемента.

grid-column: 1 / 3;

grid-row

Указывает начальную и конечную строки для элемента.

grid-row: 1 / 3;
